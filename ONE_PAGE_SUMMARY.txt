# TASD-Federation - One-Page Summary

## 🎯 Qué es esto?

Laboratorio de **federación DB2** listo para usar con Docker. Dos bases de datos que pueden hablar entre sí + acceso a archivos CSV. Compatible con Mac Arm64, Windows, Linux.

---

## ⚡ Inicio en 60 segundos

```bash
# 1. Iniciar (en el directorio del proyecto)
docker-compose up -d

# 2. Configurar (espera 30 segundos)
./setup.sh          # macOS/Linux
# o
.\setup.bat         # Windows

# 3. Conectar y usar
docker exec -it db2_federated bash
su - db2inst1
db2 connect to BASETASD
db2 "SELECT * FROM db2Lemp"     # Consultar datos remotos ✓
db2 "SELECT * FROM FILECLIENTES2"  # Consultar CSV ✓
```

## 🎓 Caso de Uso

```
Alumno ejecuta:
  db2 "SELECT E.*,D.DEPTNAME 
       FROM db2Lemp E 
       JOIN db2Ldept D ON E.WORKDEPT=D.DEPTNO"

¿Qué pasa?
  1. Query va al federador (db2_federated)
  2. El federador se conecta remotamente al servidor (db2_remote)
  3. El servidor ejecuta la consulta en la tabla real EMPLOYEE
  4. Resultados retornan al alumno ✓

⚡ Transparente: Parece una tabla local pero es remota!
```
